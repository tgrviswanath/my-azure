## Key Concepts

| Concept | Explanation |
|---------|-------------|
| NSG (Network Security Group) | Azure's stateful firewall вҖ” rules apply to inbound/outbound traffic |
| B1s VM | 1 vCPU, 1 GB RAM вҖ” cheapest general-purpose VM, good for learning |
| Public IP | Static IP assigned to the VM's NIC for external access |
| Virtual Network | Isolated Layer 3 network in Azure вҖ” like a VPC in AWS |
| Subnet | Subdivision of the VNet вҖ” VMs are placed in subnets |
| cloud-init | First-boot automation via `custom_data` in Terraform |
| Managed Disk | Azure-managed OS disk вҖ” Standard HDD is cheapest option |
