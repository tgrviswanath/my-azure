## Key Concepts

| Concept | Explanation |
|---------|-------------|
| Azurite | Official Microsoft Azure Storage emulator (replaces legacy Azure Storage Emulator) |
| Connection string | `UseDevelopmentStorage=true` redirects SDK calls to localhost |
| Port 10000 | Blob Storage endpoint вҖ” stores files/objects |
| Port 10001 | Queue Storage endpoint вҖ” async message passing |
| Port 10002 | Table Storage endpoint вҖ” NoSQL key-value store |
| Functions Core Tools | Run Azure Functions locally without any Azure subscription |
| Docker volume | Persists Azurite data between container restarts |

## Why Use Azurite Instead of Real Azure?
- No subscription required вҖ” great for CI/CD pipelines
- No network latency вҖ” faster test cycles
- No accidental charges вҖ” safe for learning
- Offline development вҖ” works without internet
- Identical API surface вҖ” code works unchanged in real Azure
