## Notes
- Azure Cost Management is completely free for Azure customers
- Budget alerts and anomaly detection are included at no charge
- Action Groups: first 1,000 email notifications/month are free
- Azure Policy (for tag enforcement): free for built-in policies

## What This Project Helps You Avoid
| Scenario | Potential Cost Without Monitoring |

| Runaway data transfer | $50-500/month |

Setting up a $20/month budget alert takes 5 minutes and can save hundreds.
